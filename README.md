
1. Clone Repository
```bash
git clone https://github.com/username/repo.git
```
2. Navigate to Repository on Local Machine
```bash
cd repo
```
3. Install Dependencies
```bash
npm install 
```
4. Run Developer Client Server
```bash
npm run dev
```
